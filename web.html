User Focused Web Design while following company Brand Guidelines

Version 1

The portal site was designed a few years ago and remained unchanged until recently. The first version of the site consisted of icons with links to new pages that contained a list of resources pertaining to the category selected. Each page opened in a new window or tab. The person in charge of the site content requested a redesign of the site since it was beginning to look dated and didn't not provide a friendly user experience.

Version 2

Version 2 was launched after a complete redesign following the brand standards of the company at the time. The colors were based on the approved color pallete. The site no longer linked out to a page for every icon to reduce the number of windows being opened by the user and imrpove site nagivation. When a user would click on an icon, the page would appear to split and slide open, then resource links would appear.

Version 2.5

Version 2.5 is the current version of the portal. It is considered 2.5 because it's release was unplanned and the update occured to bring the site up to date with the companies new brand standards. The site was migrated over to the new company custom bootstrap theme and color pallete while keeping the same functionality in order to allow users to adapt to the new look as quickly as possible.

Version 2.0

For my redesign, I used a jQuery plugin called App-Folders in order to create the page split effect when an icon was clicked. The demo to the right is functional and you can see how the page was set up to work. This type of set up provided the user with the ability to view resource links under one category without ever leaving the page.

The first version of this design showed just the icons and titles only appeared on hover. During our initial testing phase it was discovered a few users were accessing the page through Citrix on their iPads and the on hover function did not work.

Version 2.5

The latest version includes updates to the new color pallete and company Bootstrap theme. Updating to the new Bootstrap theme allowed me to clean up the code on the page and remove any code that was no longer needed. Icon images were resized to allow for proper padding as stated in the new brand guidelines.

Version 2.6

Enhancements have been requested to include an image slider at the top of the page with clickable links and add a third column to the main content section. A request was made for the site to be more interactive and eventually add in blog type capabilities as well as comments and likes. This wireframe is the first step in updating the site to include the new features. The icons have been shrunk in size to accomodate for the third column and a div placeholder has been added to the right column images or videos to be placed in.
